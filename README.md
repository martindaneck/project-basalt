# Project Basalt

This is a small renderer application made with rust, opengl and ImGUI made to be easily extensible into any sort of game / visual project you can think of. 
This is not a full engine, nor meant to be one, has a very simple file structure, it is meant to be lightweight and easily modifiable and it functions as a sort of demo.
It's a continuous work in process, it's in a usable and clean state now however it is not finished and probably ever won't be.

Features:
- OpenGL 4.6
- ImGUI
- gltf model loading
- PBR lighting
- IBL
- SSAO

To Implement:
